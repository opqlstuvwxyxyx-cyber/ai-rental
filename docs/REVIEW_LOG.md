# AI RENTAL レビュー・壁打ちログ

> Claude Code と Cursor の会話・判断・採点を記録する。  
> 毎ラウンド終了時に追記し、次の作業者は最新ラウンドから読む。

---

## 使い方

1. 新ラウンド開始時に「Round N」セクションを追加
2. Claude Code → 提案・採点を記入
3. Cursor → 実装内容・所感を記入
4. **ユーザー** → 採用/却下/次の指示を記入
5. 納得いくまで Round を繰り返す

## クイックプロンプト（コピペ用）

### Claude Code に渡す（レビュー）

```
docs/DESIGN_BRIEF.md と docs/REVIEW_LOG.md を読んで、
AI RENTAL の [ページ名] をレビューして。
コードはまだ変更しないで。
採点（10点満点）+ P0/P1/P2 + 改善案3つを出して。
結果は REVIEW_LOG.md の Round N に追記できる形式で。
```

### Cursor に渡す（実装）

```
docs/DESIGN_BRIEF.md と docs/REVIEW_LOG.md の Round N を読んで、
Claude Code の提案 [A/B/C] を実装して。
既存のデザインシステム（globals.css）に合わせること。
完了後 REVIEW_LOG.md の Cursor 欄を更新して。
```

### ユーザー判断テンプレ

```
Round N 判断:
- 採用: 案B
- NG: カードが大きすぎる
- 次: /settings を Round N+1 のテーマに
```
